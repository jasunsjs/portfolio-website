## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
# Clone and navigate to the repo
git clone https://github.com/jasunsjs/portfolio-website.git
cd portfolio-website

# Install dependencies
npm install

# Start the development server
npm run dev
```

Navigate to `http://localhost:3000` on your browser and you can go from there.
